```
│  │ ① Item one text   │  │  Y: 200+, each item:
│  │   (wrapped if long)│  │    - circled number badge (40x40, orange fill)
│  │ ② Item two text   │  │    - body text 26px beside badge
│  │ ③ Item three      │  │    - 30px vertical gap between items
│  │ ④ Item four       │  │
│  │ ⑤ Item five       │  │  Max 5 items recommended
│  ├───────────────────┤  │
│  │ [sector icon]     │  │  Optional: bottom-left
│  │              [LOGO]│  │  PUM watermark bottom-right
│  └───────────────────┘  │
└─────────────────────────┘
```

**render() method implementation:**

1. **Background:** Call `self.random_gradient()`, apply gradient. Add subtle diagonal line pattern via `self.add_diagonal_lines()` for visual texture.

2. **Title:** Use `self.get_font("heading", 44)`. Wrap if needed. Draw at Y=100, X=MARGIN. Color: white or dark_green based on background brightness (same logic as Quote/Story).

3. **Numbered items:** For each item in `data["items"]`:
   a. **Number badge:** Draw a filled circle (40px diameter) using `draw.ellipse()`. Fill color: orange (#FF6900). Draw the number (1-based) centered in the circle using `draw.text()` with `anchor="mm"`. Number font: heading Bold at 22px, color: white.
   b. **Item text:** Use `self.get_font("body", 26)`. Wrap text with `self.wrap_text(item, font, CONTENT_WIDTH - 60)` (60px offset for badge + gap). Draw starting at X = MARGIN + 60 (right of badge). Y position = current_y aligned with badge center.
   c. **Spacing:** After each item (including its wrapped lines), advance current_y by item_height + 30px gap.
   d. **Adaptive sizing:** If total items height exceeds available space (Y 200 to 900), reduce body font by 2px and item spacing by 5px, re-calculate. Minimum body font: 20px, minimum spacing: 15px.

4. **Sector icon (optional):** Same approach as Quote/Story — load from config, resize to 60x60, bottom-left.

5. **Decoration:** Add KrabbelBabbel decoration at reduced opacity (15%) in a non-text area (e.g., top-right corner).

6. **Watermark:** Call `self.add_watermark()` with appropriate logo variant.

7. **Return** the composed Image.

**Update `templates/__init__.py`:** Add `from .tips_list import TipsListTemplate`.

**IMPORTANT:** The number badge should be visually clean — a solid orange circle with a white number centered inside. Use `draw.ellipse([(x, y), (x+40, y+40)], fill=orange)` then `draw.text((x+20, y+20), str(n), font=num_font, fill="white", anchor="mm")`.
