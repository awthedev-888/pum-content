**__init__(self, config_path=None):**
- Set `self.project_root = Path(__file__).resolve().parent.parent`
- Load `brand_config.yaml` via yaml.safe_load (default path: `self.project_root / "brand_config.yaml"`)
- Call `self._load_fonts()` and `self._load_logos()` and `self._load_decorations()`

**_load_fonts(self):**
- Load heading font: `ImageFont.truetype(heading_path, size=FONT_SIZES["heading_medium"])`
- After loading, call `self.font_heading.set_variation_by_axes([700, 100])` to set Bold weight (this is CRITICAL — NotoSans-Bold.ttf is a variable font renamed, defaults to 400 without this)
- Load body font: `ImageFont.truetype(body_path, size=FONT_SIZES["body_medium"])`
- After loading, call `self.font_body.set_variation_by_axes([400, 100])` to set Regular weight
- Load decorative font: `ImageFont.truetype(decorative_path, size=FONT_SIZES["decorative"])`
- Store as `self.font_heading`, `self.font_body`, `self.font_decorative`

**_load_logos(self):**
- Load primary logo (dark green) from config path: `Image.open(...).convert("RGBA")`
- Load white logo from config path: `Image.open(...).convert("RGBA")`
- Store as `self.logo_primary`, `self.logo_white`

**_load_decorations(self):**
- Load the 3 KrabbelBabbel PNGs from the decorations directory in config
- Store as `self.decorations` list of RGBA Images

**create_canvas(self, bg_color="#D2E8D7") -> Image.Image:**
- Return `Image.new("RGBA", (self.WIDTH, self.HEIGHT), bg_color)`

**draw_gradient(self, img, color1, color2, direction="vertical") -> Image.Image:**
- Use line strip paste method (NOT putpixel — too slow)
- For vertical: iterate `y` from 0 to HEIGHT, interpolate RGB, create 1px-high strip via `Image.new("RGB", (WIDTH, 1), (r, g, b))`, paste at `(0, y)`
- For horizontal: similar but iterate `x` and paste 1px-wide strips
- Return modified image

**random_gradient(self) -> tuple[str, str]:**
- Return `random.choice(self.GRADIENT_COMBOS)`

**add_dot_pattern(self, img, color_rgba=(255, 255, 255, 30), spacing=60) -> Image.Image:**
- Create RGBA overlay same size as canvas, fully transparent
- Draw small ellipses (radius 3px) at regular grid spacing
- Use `Image.alpha_composite(img, overlay)` to merge
- Return composited image

**add_diagonal_lines(self, img, color_rgba=(255, 255, 255, 20), spacing=40, width=1) -> Image.Image:**
- Create RGBA overlay
- Draw diagonal lines from top-left to bottom-right at regular spacing
- Use `Image.alpha_composite()` to merge
- Return composited image

**add_decoration(self, img, deco_index=0, position=(0, 0), size=200, opacity=0.2) -> Image.Image:**
- Get decoration from `self.decorations[deco_index]`
- Resize maintaining aspect ratio using `Image.LANCZOS`
- Adjust alpha: split RGBA, apply `a.point(lambda x: int(x * opacity))`, re-merge
- Paste onto img using alpha mask: `img.paste(deco, position, deco)`
- Return modified image

**add_watermark(self, img, padding=None, width=None, use_white=False) -> Image.Image:**
- Default padding = self.WATERMARK_PADDING, width = self.WATERMARK_WIDTH
- Select logo: `self.logo_white if use_white else self.logo_primary`
- Calculate aspect ratio, resize with `Image.LANCZOS`
- Position: bottom-right corner with padding
- Paste with alpha mask: `img.paste(logo, pos, logo)`
- Return modified image

**wrap_text(self, text, font, max_width) -> list[str]:**
- Split by words, use `font.getlength(test_line)` for pixel measurement
- Build lines word-by-word, break when line exceeds max_width
- Handle single words longer than max_width (add them as their own line)
- Return list of strings

**get_text_block_height(self, lines, font, line_spacing=10) -> int:**
- Use `font.getbbox("Ag")[3]` for line height (captures ascender + descender)
- Return `len(lines) * line_height + (len(lines) - 1) * line_spacing`

**get_font(self, style, size) -> ImageFont.FreeTypeFont:**
- Helper to load a font at a specific size from brand config
- style: "heading", "body", or "decorative"
- Load from the same file path, at the requested size
- For heading: call `set_variation_by_axes([700, 100])` after loading
- For body: call `set_variation_by_axes([400, 100])` after loading
- Return the font object

**get_color(self, group, name) -> str:**
- Return `self.config["colors"][group][name]`

**render(self, data) -> Image.Image:**
- Raise `NotImplementedError("Subclasses must implement render()")`

**save(self, img, output_path):**
- Create parent directories with `output_path.parent.mkdir(parents=True, exist_ok=True)`
- Save as PNG: `img.save(str(output_path), "PNG")`

**IMPORTANT:** Do NOT use deprecated Pillow APIs:
- Use `font.getbbox()` NOT `font.getsize()` (removed in Pillow 10.0)
- Use `font.getlength()` NOT `draw.textsize()` (removed in Pillow 10.0)
- Use `Image.LANCZOS` NOT `Image.ANTIALIAS` (removed in Pillow 10.0)
- Use `draw.rounded_rectangle()` where applicable (available since Pillow 8.2)
