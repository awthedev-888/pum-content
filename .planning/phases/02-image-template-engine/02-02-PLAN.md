```
│  ├───────────────────┤  │
│  │   Body text        │  │  Y: 260-700, font: body 28px Regular
│  │   (multi-line      │  │  Max ~15 lines
│  │    wrapped)         │  │
│  ├───────────────────┤  │
│  │   — Attribution    │  │  Y: after body + gap, font: body 22px
│  ├───────────────────┤  │
│  │ [KrabbelBabbel]   │  │  Decorative accent (15-20% opacity)
│  │              [LOGO]│  │  PUM watermark bottom-right
│  └───────────────────┘  │
└─────────────────────────┘
```

**render() method implementation:**

1. **Background:** Call `self.random_gradient()` to get color pair, then `self.draw_gradient(canvas, c1, c2)`. Add subtle dot pattern via `self.add_dot_pattern()`.

2. **Headline:** Use `self.get_font("heading", 48)` for headline. Wrap with `self.wrap_text(headline, font, CONTENT_WIDTH)`. Draw at Y=120, X=MARGIN. Color: white (#FFFFFF) for dark-topped gradients.

3. **Divider:** Draw a horizontal line at Y = headline_bottom + 20, from X=MARGIN to X=MARGIN+200. Color: orange (#FF6900), width 3px. Use `draw.line()`.

4. **Body text:** Use `self.get_font("body", 28)` for body. Wrap text. Calculate available height between divider and attribution zone. **Adaptive sizing:** If wrapped text exceeds available height, reduce font size by 2px and re-wrap, down to minimum 20px. Draw each line with 10px line spacing. Color: white or dark green depending on gradient midpoint brightness.

5. **Attribution:** Use `self.get_font("body", 22)` for attribution. Draw below body text + 30px gap. Color: light beige (#F8E3B3) or soft gold (#E9C779).

6. **Sector icon (optional):** If `data.get("sector")` is provided and exists in brand config icons, load the icon PNG, resize to 60x60, place at bottom-left (X=MARGIN, Y=HEIGHT-MARGIN-60). Use alpha mask for paste.

7. **Decoration:** Call `self.add_decoration()` with a random KrabbelBabbel at ~15% opacity, positioned in a corner that doesn't overlap text (e.g., bottom-left or top-right).

8. **Watermark:** Call `self.add_watermark(img, use_white=use_white)` — use white logo on dark backgrounds, primary logo on light backgrounds. Determine by checking first gradient color brightness.

9. **Return** the composed Image.

**Text color logic:** For dark gradient tops (dark_green #0E5555), use white text. For light gradient tops (orange #FF6900, blue #659BD1), use dark_green text. Implement a simple brightness check: `sum(ImageColor.getrgb(color)) / 3 < 128` → dark → white text.

**Update `templates/__init__.py`:** Add `from .quote_story import QuoteStoryTemplate`.

**IMPORTANT:** Do NOT use deprecated APIs. Use `font.getbbox()` for bounding boxes, `font.getlength()` for horizontal width, `Image.LANCZOS` for resampling, `draw.text()` with `anchor` parameter for alignment where useful.
