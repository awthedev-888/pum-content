**Data contract (render input):**
```python
data = {
    "title": str,               # e.g. "Dampak PUM di Indonesia"

│  ├───────────────────┤  │
│  │                    │  │
│  │    1.200+          │  │  Y: ~250, font: heading 96px Bold, ORANGE
│  │  ahli sukarelawan  │  │  Y: ~360, font: body 28px
│  │                    │  │
│  │      30+           │  │  Y: ~440, second stat
│  │    negara aktif    │  │
│  │                    │  │
│  │      45+           │  │  Y: ~600, third stat (optional)
│  │   tahun pengalaman │  │
│  ├───────────────────┤  │
│  │ [decoration]      │  │
│  │              [LOGO]│  │  PUM watermark bottom-right
│  └───────────────────┘  │
└─────────────────────────┘
```

**render() method implementation:**

1. **Background:** Call `self.random_gradient()`, apply gradient. Add dot pattern via `self.add_dot_pattern()` with very subtle opacity (alpha 20).

2. **Title:** Use `self.get_font("heading", 44)`. Wrap if needed. Draw at Y=100, X=MARGIN, centered horizontally using `anchor="mt"` (middle-top) with X at canvas center (540). Color: white or dark_green based on background brightness.

3. **Stats section:** Calculate available vertical space for stats: Y range 220 to 880. Divide evenly among `len(data["stats"])` stats.

   For each stat:
   a. **Number:** Use `self.get_font("heading", 96)` (or 80 for 3 stats, to fit). Call `set_variation_by_axes([700, 100])` for Bold. Draw centered horizontally (`anchor="mt"`, X=540). Color: orange (#FF6900) — this is the visual anchor.
   b. **Label:** Use `self.get_font("body", 28)`. Draw centered below number with 15px gap. Color: white or light_beige depending on background.
   c. **Divider (optional):** Between stats, draw a small centered horizontal line (100px wide, 1px, low opacity) as a subtle separator. Use soft_gold (#E9C779).

4. **Adaptive stat sizing:** If 1 stat: use large font (96px number), generous vertical centering. If 2 stats: 80px numbers, split space evenly. If 3 stats: 72px numbers, tighter spacing.

5. **Decoration:** Add KrabbelBabbel at low opacity (15%) in bottom-left area.

6. **Watermark:** Call `self.add_watermark()`.

7. **Return** the composed Image.

**Update `templates/__init__.py`:** Add `from .impact_stats import ImpactStatsTemplate`.

**Key visual principle:** The large orange numbers must be the dominant visual element. They should be significantly larger than all other text. Use center alignment for the entire stats block to create a clean, bold look.
