   - Wrap entire function in try/except: log warning on any exception, return ""

   - If `feed.entries` is empty, return ""
   - For each entry (up to 5): extract title, summary, link, published
   - Format as `"### {title} ({published})\n{summary}\nSource: {link}"`
   - Join with "\n\n"
   - Wrap in try/except, return "" on failure with warning log
