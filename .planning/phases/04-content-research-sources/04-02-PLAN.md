1. Create `research_sources/content_brief.py` implementing `load_content_brief(filepath: str = "content_brief.yaml") -> str`:
   - Check `os.path.exists(filepath)`. If not found, log info and return "".
   - Open file with `encoding="utf-8"`, parse with `yaml.safe_load()`.
   - If data is None or empty dict, return "".
   - Extract three sections:
     - `story_ideas`: list of dicts with `title` and `description` -> format as "- {title}: {description}"
     - `stats`: list of dicts with `number` and `context` -> format as "- {number} {context}"
     - `events`: list of dicts with `name`, `date`, `details` -> format as "- {name} ({date}): {details}"
   - Each present section becomes a labeled block: "Story Ideas:\n...", "Key Statistics:\n...", "Upcoming Events:\n..."
   - Join non-empty sections with "\n\n"
   - Wrap in try/except, return "" on failure

3. Create `research_sources/sheets_reader.py` implementing `read_content_sheet(sheet_id: str) -> str`:
   - Check `os.environ.get("GSHEET_CREDENTIALS")`. If not set, log warning and return "".
   - Parse credentials with `json.loads()`. Handle `json.JSONDecodeError` -> log warning, return "".
   - Authenticate with `gspread.service_account_from_dict(credentials_dict)`.
   - Open spreadsheet by key: `gc.open_by_key(sheet_id)`.
   - Get first worksheet: `spreadsheet.sheet1`.
   - Get all records: `worksheet.get_all_records()` (first row = headers).
   - Format each row as "key: value" pairs (skip empty values), joined by "; ". Join rows with "\n".
   - Wrap in try/except, return "" on failure
