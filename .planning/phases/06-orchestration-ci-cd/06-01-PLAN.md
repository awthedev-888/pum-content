**Module docstring:**
```python
"""PUM Indonesia Content Generator - Main Pipeline Orchestrator.

Runs the complete content pipeline: research -> generate -> render -> email.
Designed to be called by GitHub Actions cron or manually via python main.py.
"""
```

**Imports (at module level):**
```python
import os
import sys
import logging
from datetime import date

from dotenv import load_dotenv
```

**Logger:** `logger = logging.getLogger(__name__)`

**`render_image(post) -> str` function:**
- Import template classes inside the function (deferred import to avoid import-time failures):
  ```python
  from templates import QuoteStoryTemplate, TipsListTemplate, ImpactStatsTemplate
  ```
- Define template_map dictionary:
  ```python
  template_map = {
      "quote_story": QuoteStoryTemplate,
      "tips_list": TipsListTemplate,
      "impact_stats": ImpactStatsTemplate,
  }
  ```
- Look up template_class from template_map using `post.template_type`
- If not found, raise `ValueError(f"Unknown template type: {post.template_type}")`
- Instantiate template: `template = template_class()`
- Render image: `img = template.render(post.template_data)`
- Build output path: `output_path = f"output/pum_post_{date.today().isoformat()}.png"`
- Save image: `template.save(img, output_path)` (BaseTemplate.save creates parent dirs)
- Return output_path

**`run_pipeline() -> bool` function:**
Execute the 4-stage pipeline. Each stage is wrapped in its own try/except block. If a stage fails, log the error and return False immediately (all stages are critical -- downstream stages cannot proceed without upstream output).

**Stage 1: Research**
```python
logger.info("=" * 50)
logger.info("Stage 1: Gathering source material")
logger.info("=" * 50)
try:
    from research_sources import gather_source_material
    sheet_id = os.environ.get("GOOGLE_SHEET_ID")
    source_material = gather_source_material(sheet_id=sheet_id)
    logger.info("Source material gathered: %d characters", len(source_material))
except Exception as e:
    logger.error("Stage 1 FAILED - Research: %s", e)
    return False
```

**Stage 2: Generate content**
```python
logger.info("=" * 50)
logger.info("Stage 2: Generating content with Gemini")
logger.info("=" * 50)
try:
    from content_generator import generate_post
    post = generate_post(source_material)
    logger.info(
        "Content generated: pillar=%s, template=%s, hashtags=%d",
        post.content_pillar,
        post.template_type,
        len(post.hashtags),
    )
except Exception as e:
    logger.error("Stage 2 FAILED - Generation: %s", e)
    return False
```

**Stage 3: Render image**
```python
logger.info("=" * 50)
logger.info("Stage 3: Rendering branded image")
logger.info("=" * 50)
try:
    image_path = render_image(post)
    logger.info("Image saved: %s", image_path)
except Exception as e:
    logger.error("Stage 3 FAILED - Render: %s", e)
    return False
```

**Stage 4: Send email**
```python
logger.info("=" * 50)
logger.info("Stage 4: Sending email")
logger.info("=" * 50)
try:
    from email_sender import send_post_email
    send_post_email(post, image_path)
    logger.info("Pipeline complete - email sent successfully")
except Exception as e:
    logger.error("Stage 4 FAILED - Email: %s", e)
    return False
```

Return True at the end (all stages succeeded).

**`main()` function:**
```python
def main():
    """Entry point: configure logging, load env, run pipeline."""
    logging.basicConfig(
        level=logging.INFO,
        format="%(asctime)s [%(levelname)s] %(name)s: %(message)s",
        datefmt="%Y-%m-%d %H:%M:%S",
    )

    logger.info("PUM Indonesia Content Generator - Starting pipeline")
    logger.info("Date: %s", date.today().isoformat())

    # Load .env for local development (no-op if .env doesn't exist)
    load_dotenv()

    success = run_pipeline()

    if success:
        logger.info("Pipeline completed successfully")
    else:
        logger.error("Pipeline failed - check logs above for details")

    sys.exit(0 if success else 1)
```

**Entry point guard:**
```python
if __name__ == "__main__":
    main()
```

**IMPORTANT design notes:**
- Imports of pipeline modules are deferred (inside try blocks) to prevent import-time failures from blocking error reporting.
- Only non-sensitive values are logged: character counts, pillar names, template types, hashtag counts, file paths. Never log API keys, passwords, or credential JSON.
- Each stage uses `return False` on failure (not raising) so the orchestrator can log a clean error summary.
- `run_pipeline()` returns a bool so tests can verify pipeline flow without intercepting sys.exit.
