**Tests 1-4: Pipeline stage failure isolation**

For each stage failure test, mock all 4 pipeline modules at module level using `@patch`. Configure the failing stage to raise `RuntimeError("stage X failed")` while other stages return valid mock data. Assert `run_pipeline() is False`.

- Test 1 (research fails): `mock_gather.side_effect = RuntimeError("All sources failed")` -> `run_pipeline()` returns False
- Test 2 (generation fails): `mock_gather.return_value = "source text"`, `mock_generate.side_effect = RuntimeError("API error")` -> returns False
- Test 3 (render fails): gather and generate succeed, but patch `main.render_image` to raise `ValueError("Unknown template")` -> returns False
- Test 4 (email fails): gather, generate, render succeed, `mock_send.side_effect = RuntimeError("SMTP error")` -> returns False

**Mock strategy for pipeline tests:**
The orchestrator uses deferred imports (inside try blocks), so we need to mock at the correct module level:
```python
# For research stage:
@patch("research_sources.gather_source_material")

# For generation stage:
@patch("content_generator.generate_post")

# For render stage (mock render_image in main module):
@patch("main.render_image")

# For email stage:
@patch("email_sender.send_post_email")
```

Note: Since `run_pipeline()` uses `from research_sources import gather_source_material` inside the function, we need to mock the actual module functions. The deferred imports mean we mock the source modules, not main module references. For `render_image`, since it's called directly (not imported), mock `main.render_image`.

**Error accumulator pattern:**
```python
def main():
    errors = []
    # ... each test wrapped in try/except, appends to errors on failure ...

    total_tests = N
    passed = total_tests - len(errors)
    print(f"\nMain Pipeline Tests: {passed}/{total_tests} passed")

    if errors:
        print(f"\nFAILED: {len(errors)} error(s)")
        for err in errors:
            print(f"  - {err}")
        sys.exit(1)
    else:
        print("All tests passed!")
        sys.exit(0)

if __name__ == "__main__":
    main()
```

**IMPORTANT mock notes:**
- Use `@patch` decorators or `with patch(...)` context managers -- both work with error accumulator
- Since `run_pipeline()` imports modules inside the function body, mock the actual module paths (e.g., `research_sources.gather_source_material`), not `main.gather_source_material`
- For `render_image`, mock `main.render_image` when testing pipeline flow, but mock template classes when testing render_image itself
- Configure `logging.disable(logging.CRITICAL)` at top of main() to suppress log output during tests, then `logging.disable(logging.NOTSET)` at end
