<task type="auto">
  <name>Task 2: Download Google Fonts (Noto Sans and Permanent Marker) as static TTF files</name>
  <files>assets/fonts/NotoSans-Regular.ttf, assets/fonts/NotoSans-Bold.ttf, assets/fonts/PermanentMarker-Regular.ttf</files>
  <action>
Download font TTF files from Google Fonts. Use STATIC font files (not variable fonts) for maximum Pillow compatibility — the research notes variable font API differences across Pillow versions. Static TTFs are the safe choice.

1. Download Noto Sans static TTFs from Google Fonts GitHub repo:
```bash
# Noto Sans Regular
curl -L -o assets/fonts/NotoSans-Regular.ttf "https://github.com/google/fonts/raw/main/ofl/notosans/NotoSans%5Bwdth%2Cwght%5D.ttf" 2>/dev/null

# If variable font downloaded, we need static versions instead.
# Try the Google Fonts API download approach:
curl -L -o /tmp/NotoSans.zip "https://fonts.google.com/download?family=Noto+Sans" 2>/dev/null
# Extract static Regular and Bold from the ZIP
unzip -j /tmp/NotoSans.zip "static/NotoSans-Regular.ttf" -d assets/fonts/ 2>/dev/null
unzip -j /tmp/NotoSans.zip "static/NotoSans-Bold.ttf" -d assets/fonts/ 2>/dev/null
```

If the static folder is not present in the ZIP, use the variable font file and rename it:
```bash
unzip -j /tmp/NotoSans.zip "NotoSans-VariableFont_wdth,wght.ttf" -d assets/fonts/ 2>/dev/null
# Rename for consistency
mv assets/fonts/NotoSans-VariableFont_wdth,wght.ttf assets/fonts/NotoSans-Regular.ttf 2>/dev/null
# For bold, we'll use the same file at runtime with weight axis set to 700
cp assets/fonts/NotoSans-Regular.ttf assets/fonts/NotoSans-Bold.ttf 2>/dev/null
```

2. Download Permanent Marker:
```bash
curl -L -o /tmp/PermanentMarker.zip "https://fonts.google.com/download?family=Permanent+Marker" 2>/dev/null
unzip -j /tmp/PermanentMarker.zip "PermanentMarker-Regular.ttf" -d assets/fonts/ 2>/dev/null
```

Alternative direct download from GitHub if ZIP approach fails:
```bash
curl -L -o assets/fonts/PermanentMarker-Regular.ttf "https://github.com/google/fonts/raw/main/apache/permanentmarker/PermanentMarker-Regular.ttf"
```

3. Verify all fonts load in Pillow:
```python
from PIL import ImageFont
from pathlib import Path

fonts_dir = Path("assets/fonts")

# Test each font loads and can render text
for font_file in ["NotoSans-Regular.ttf", "NotoSans-Bold.ttf", "PermanentMarker-Regular.ttf"]:
    path = fonts_dir / font_file
    assert path.exists(), f"Missing: {path}"
    font = ImageFont.truetype(str(path), size=24)
    # Test rendering Indonesian text
    bbox = font.getbbox("PUM Indonesia — Bersama Kita Tumbuh")
    assert bbox is not None, f"Font {font_file} cannot render text"
    print(f"OK: {font_file} loaded, bbox={bbox}")
```

IMPORTANT: If static Noto Sans files are not available and variable font is used instead, note this in the summary. The brand_config.yaml in Plan 01-03 will reference the actual filenames present in assets/fonts/.

Decision (Claude's discretion per CONTEXT.md): Using static TTF files instead of variable fonts for maximum Pillow compatibility. Research noted the variable font axis API has changed between Pillow versions — static files avoid this risk entirely.
