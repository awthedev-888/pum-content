1. Create `email_sender/composer.py` implementing three functions:

   **Imports:**
   ```python
   import os
   import logging
   from datetime import date
   from email.mime.multipart import MIMEMultipart
   from email.mime.text import MIMEText
   from email.mime.image import MIMEImage
   from email.utils import formatdate, formataddr

   from email_sender.smtp_client import send_email
   ```

   **Logger:** `logger = logging.getLogger(__name__)`

   **`format_email_body(post) -> str`:**
   - Takes any object with attributes: `caption_id`, `caption_en`, `hashtags` (list[str]), `posting_suggestion`, `content_pillar`, `template_type`
   - Format hashtags: `" ".join(f"#{tag}" for tag in post.hashtags)`
   - Build plain text body with clear copy-paste sections using this exact format:
     ```
     PUM Indonesia - Daily Instagram Post
     ========================================

     POSTING SUGGESTION
     --------------------
     {post.posting_suggestion}

     Content pillar: {post.content_pillar}
     Template type: {post.template_type}

     CAPTION (Bahasa Indonesia)
     --------------------
     {post.caption_id}

     CAPTION (English)
     --------------------
     {post.caption_en}

     HASHTAGS
     --------------------
     {hashtags_str}

     ========================================
     Image attached: pum_post_{date.today().isoformat()}.png

     Generated by PUM Content Generator
     ```
   - Use f-string with the exact layout above. The `{'=' * 40}` and `{'-' * 20}` separators make sections visually distinct for copy-paste.

   **`compose_email(post, image_path, from_addr, to_addr) -> MIMEMultipart`:**
   - Validate image exists: `if not os.path.isfile(image_path): raise FileNotFoundError(f"Image not found: {image_path}")`
   - Validate image not empty: `if os.path.getsize(image_path) == 0: raise FileNotFoundError(f"Image file is empty: {image_path}")`
   - Create message: `msg = MIMEMultipart("mixed")`
   - Set headers:
     - `msg["From"] = formataddr(("PUM Content Generator", from_addr))`
     - `msg["To"] = to_addr`
     - `msg["Date"] = formatdate(localtime=True)`
     - `msg["Subject"] = f"PUM Instagram Post - {date.today().isoformat()}"` (e.g., "PUM Instagram Post - 2026-03-01")
   - Attach text body: `msg.attach(MIMEText(format_email_body(post), "plain", "utf-8"))` — always utf-8 for Bahasa Indonesia character support. Do NOT omit the charset parameter.
   - Attach image:
     ```python
     with open(image_path, "rb") as f:
         img_part = MIMEImage(f.read(), _subtype="png")
     img_part.add_header(
         "Content-Disposition", "attachment",
         filename=f"pum_post_{date.today().isoformat()}.png"
     )
     msg.attach(img_part)
     ```
   - Return `msg`

   **`send_post_email(post, image_path) -> None`:**
   - This is the top-level convenience function that Phase 6 orchestrator calls
   - Read three env vars: `GMAIL_ADDRESS`, `GMAIL_APP_PASSWORD`, `RECIPIENT_EMAIL` from `os.environ.get()`
   - Collect missing ones in a list. If any missing, raise `ValueError(f"Missing environment variables: {', '.join(missing)}")`
   - Compose message: `msg = compose_email(post, image_path, gmail_address, recipient_email)`
   - Send message: `send_email(msg)` (imported from smtp_client)
   - Log: `logger.info("Post email sent to %s", recipient_email)`
