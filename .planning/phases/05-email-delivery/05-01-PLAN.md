   **Imports:**
   ```python
   import os
   import ssl
   import smtplib
   import logging
   ```

   **Constants:**
   ```python
   GMAIL_SMTP_HOST = "smtp.gmail.com"
   GMAIL_SMTP_PORT = 587
   SMTP_TIMEOUT = 30
   ```

   **Logger:** `logger = logging.getLogger(__name__)`

   **`send_email(msg) -> None` function:**
   - Read `GMAIL_ADDRESS` from `os.environ.get("GMAIL_ADDRESS")`
   - Read `GMAIL_APP_PASSWORD` from `os.environ.get("GMAIL_APP_PASSWORD")`
   - Collect missing variable names in a list. If any missing, raise `ValueError(f"Missing environment variables: {', '.join(missing)}")`
   - Strip spaces from App Password: `gmail_password = gmail_password.replace(" ", "")` — Google displays App Passwords as "abcd efgh ijkl mnop" with spaces for readability, but the actual password is the 16 chars without spaces
   - Create secure SSL context: `context = ssl.create_default_context()` — loads system CA certs, enables hostname checking
   - Open SMTP connection as context manager:
     ```python
     with smtplib.SMTP(GMAIL_SMTP_HOST, GMAIL_SMTP_PORT, timeout=SMTP_TIMEOUT) as server:
         server.starttls(context=context)
         server.login(gmail_address, gmail_password)
         server.send_message(msg)
     ```
   - Log success: `logger.info("Email sent successfully to %s", msg["To"])`
   - Do NOT catch `smtplib.SMTPAuthenticationError` or other SMTP exceptions — let them propagate to the caller. The Phase 6 orchestrator handles error reporting.

   **Docstring for send_email:**
   ```python
   def send_email(msg):
       """Send a composed email message via Gmail SMTP.

       Connects to smtp.gmail.com:587 with STARTTLS encryption and
       authenticates using App Password from environment variables.

       Args:
           msg: email.mime.multipart.MIMEMultipart message object.

       Raises:
           ValueError: If GMAIL_ADDRESS or GMAIL_APP_PASSWORD not set.
           smtplib.SMTPAuthenticationError: If credentials are invalid.
           smtplib.SMTPException: If send fails after authentication.
       """
   ```

**Helper for environment management:** Save and restore env vars in each test to prevent pollution:
```python
def clear_email_env():
    """Remove email env vars if set, return originals for restore."""
    saved = {}
    for key in ["GMAIL_ADDRESS", "GMAIL_APP_PASSWORD", "RECIPIENT_EMAIL"]:
        saved[key] = os.environ.pop(key, None)
    return saved

def restore_env(saved):
    """Restore previously saved env vars."""
    for key, val in saved.items():
        if val is not None:
            os.environ[key] = val
        else:
            os.environ.pop(key, None)
```
